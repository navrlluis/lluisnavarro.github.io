# Estructura para GitHub Pages Multiidioma ## Estructura de carpetas recomendada: ``` tu-repositorio/ ├── index.html # Página de redirección principal ├── _config.yml # Configuración de Jekyll (opcional) ├── es/ │ ├── index.html # Página principal en español │ ├── about.html # Otras páginas en español │ └── ... ├── en/ │ ├── index.html # Página principal en inglés │ ├── about.html # Otras páginas en inglés │ └── ... └── assets/ ├── css/ ├── js/ └── images/ ``` ## Pasos para configurar: ### 1. Crear el repositorio - Ve a GitHub y crea un nuevo repositorio - Nombre sugerido: `tu-usuario.github.io` (para sitio principal) o `nombre-proyecto` (para sitio de proyecto) ### 2. Configurar GitHub Pages - Ve a Settings > Pages - En "Source" selecciona "Deploy from a branch" - Selecciona "main" branch y "/ (root)" ### 3. Crear index.html principal (redirección) ```html Redirigiendo...

Redirigiendo a tu idioma preferido...

Español | English

``` ### 4. Crear /es/index.html (página principal) ```html Mi Sitio Web
Inicio English

Bienvenido a mi sitio web

Esta es la página principal en español.

``` ### 5. Crear /en/index.html ```html My Website
Home Español

Welcome to my website

This is the main page in English.

``` ## Comandos Git para subir: ```bash # Clonar tu repositorio o inicializar git clone https://github.com/tu-usuario/tu-repositorio.git cd tu-repositorio # Crear la estructura de archivos mkdir es en assets # Añadir archivos git add . git commit -m "Initial multilingual site structure" git push origin main ``` ## URLs resultantes: - **Sitio principal**: `https://tu-usuario.github.io/tu-repositorio/` - **Versión española**: `https://tu-usuario.github.io/tu-repositorio/es/` - **Versión inglesa**: `https://tu-usuario.github.io/tu-repositorio/en/` ## Consejos adicionales: 1. **Configuración Jekyll** (_config.yml): ```yaml title: Mi Sitio Web description: Sitio web multiidioma baseurl: "/tu-repositorio" url: "https://tu-usuario.github.io" plugins: - jekyll-sitemap - jekyll-feed ``` 2. **Para sitio personal principal**: Si usas `tu-usuario.github.io` como nombre del repositorio, no necesitas `/tu-repositorio/` en las URLs. 3. **Navegación consistente**: Mantén enlaces de cambio de idioma en todas las páginas. 4. **SEO multiidioma**: Usa etiquetas `hreflang` para mejor SEO: ```html ```
